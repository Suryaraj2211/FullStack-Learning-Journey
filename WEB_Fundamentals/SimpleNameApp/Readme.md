# Simple Name App

This is a basic JavaScript DOM project where the user enters a name in an input box and displays it on the screen by clicking a button.

---

## Project Description

The Simple Name App allows users to:
- Type text in an input box
- Click a button
- Instantly display the entered text on the screen using JavaScript

This project is created to practice basic DOM manipulation and event handling.

---
## Features

- Text input field
- Button click interaction
- Live display of user input after button click
- Clean and minimal UI
- No page reload required

---

## Concepts Used

- HTML input elements
- JavaScript DOM selection using `querySelector`
- `addEventListener`
- `click` event
- `innerText` for updating UI dynamically
